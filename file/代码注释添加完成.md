## 🎯 注释内容示例

### AuthServiceImpl 注释示例

```java
/**
 * 用户注册
 * 1. 检查用户名是否已存在
 * 2. 检查邮箱是否已存在
 * 3. 密码加密存储
 * 4. 生成 JWT Token
 */
@Override
public LoginResponse register(RegisterRequest request) {
    if (userRepository.existsByUsername(request.getUsername())) {
        throw new RuntimeException("用户名已存在");
    }
    if (userRepository.existsByEmail(request.getEmail())) {
        throw new RuntimeException("邮箱已存在");
    }

    // 创建新用户，密码使用 BCrypt 加密
    User user = User.builder()
            .username(request.getUsername())
            .email(request.getEmail())
            .password(passwordEncoder.encode(request.getPassword()))
            .phone(request.getPhone())
            .build();

    user = userRepository.save(user);
    String token = jwtTokenProvider.generateToken(user.getId());

    return LoginResponse.builder()
            .token(token)
            .user(convertToDTO(user))
            .build();
}
```

### HotelServiceImpl 注释示例

```java
/**
 * 搜索酒店
 * 支持按城市或关键词搜索
 */
@Override
public PageResponse<Hotel> searchHotels(String city, String keyword, Integer page, Integer pageSize) {
    Pageable pageable = PageRequest.of(page - 1, pageSize);
    Page<Hotel> result;

    if (city != null && !city.isEmpty()) {
        // 按城市搜索
        result = hotelRepository.findByCity(city, pageable);
    } else if (keyword != null && !keyword.isEmpty()) {
        // 按关键词搜索
        result = hotelRepository.findByNameContaining(keyword, pageable);
    } else {
        // 获取所有酒店
        result = hotelRepository.findAll(pageable);
    }

    return PageResponse.<Hotel>builder()
            .list(result.getContent())
            .total(result.getTotalElements())
            .page(page)
            .pageSize(pageSize)
            .build();
}
```

## 📚 注释标准

### 接口注释标准
- 类级注释：说明接口的功能和用途
- 方法注释：说明方法的功能、参数、返回值

### 实现类注释标准
- 类级注释：说明实现类的功能和用途
- 方法注释：说明方法的功能、参数、返回值、实现细节
- 代码行注释：说明关键业务逻辑

✅ **便于文档生成** - 可以使用 JavaDoc 工具生成 API 文档
