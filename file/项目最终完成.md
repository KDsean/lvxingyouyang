## 🚀 快速启动

### 前置要求
- Java 17+
- Maven 3.6+
- MySQL 8.0+

### 启动步骤
```bash
# 1. 初始化数据库
mysql -u root -p < init.sql

# 2. 修改配置文件
# 编辑 src/main/resources/application.yml

# 3. 编译项目
mvn clean install

# 4. 运行应用
mvn spring-boot:run

# 5. 验证应用
# 访问 http://localhost:8080/api/hotels/popular
```
