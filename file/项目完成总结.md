## API 端点总览

### 认证模块 (12 个端点)
- POST /auth/register - 注册
- POST /auth/login - 登录
- GET /auth/user - 获取用户信息
- PUT /auth/profile - 更新用户信息
- POST /auth/logout - 登出

## 快速启动

### 1. 数据库初始化
```bash
mysql -u root -p < init.sql
```

### 2. 修改配置
编辑 `application.yml` 中的数据库连接信息

### 3. 编译运行
```bash
mvn clean install
mvn spring-boot:run
```

### 4. 验证
访问 `http://localhost:8080/api/hotels/popular`
