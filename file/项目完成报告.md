## 🎯 功能实现

### 12 个核心业务模块

#### 1. 认证系统 (AuthService)
- 用户注册
- 用户登录
- 获取当前用户信息
- 更新用户信息
- 用户登出

#### 2. 酒店预订 (HotelService)
- 搜索酒店
- 获取酒店详情
- 获取推荐酒店
- 获取热门酒店
- 预订酒店

#### 3. 机票预订 (FlightService)
- 搜索航班
- 获取航班详情
- 获取热门航线
- 预订机票

#### 4. 火车票 (TrainService)
- 搜索火车票
- 获取火车详情
- 获取车站列表
- 预订火车票

#### 5. 租车自驾 (CarRentalService)
- 搜索租车
- 获取租车详情
- 获取热门地点
- 预订租车

#### 6. 景点门票 (AttractionService)
- 搜索景点
- 获取景点详情
- 获取热门景点
- 获取推荐景点
- 购买门票

#### 7. 当地攻略 (GuideService)
- 获取攻略列表
- 获取攻略详情
- 发布攻略
- 点赞攻略
- 获取热门攻略
- 搜索攻略

#### 8. 美食林 (RestaurantService)
- 搜索餐厅
- 获取餐厅详情
- 获取热门餐厅
- 获取推荐餐厅
- 预订餐厅

#### 9. 全球购 (ProductService)
- 获取商品列表
- 获取商品详情
- 获取热门商品
- 获取推荐商品
- 添加到购物车
- 获取购物车
- 创建订单

#### 10. 目的地探索 (DestinationService)
- 获取目的地列表
- 获取目的地详情
- 获取热门目的地
- 搜索目的地
- 获取热力图数据

#### 11. AI 行程规划 (TripPlanService)
- 获取我的行程
- 获取行程详情
- 创建行程
- 更新行程
- 删除行程

#### 12. 订单管理 (OrderService)
- 获取我的订单
- 获取订单详情
- 取消订单

## 🔐 安全特性

### 认证方式
- JWT Token 认证
- Bearer Token 在请求头中传递
- Token 自动过期机制

### 密码安全
- BCrypt 密码加密
- 密码不可逆存储
- 登录时密码验证

### 授权控制
- 基于 Spring Security 的权限控制
- 某些接口需要认证
- 用户只能访问自己的数据

### 跨域支持
- CORS 配置完整
- 支持前端跨域请求
- 支持预检请求

## 🚀 快速启动

### 环境要求
- Java 17+
- Maven 3.6+
- MySQL 8.0+

### 启动步骤
```bash
# 1. 初始化数据库
mysql -u root -p < init.sql

# 2. 修改配置文件
# 编辑 src/main/resources/application.yml

# 3. 编译项目
mvn clean install

# 4. 运行应用
mvn spring-boot:run

# 5. 验证应用
# 访问 http://localhost:8080/api/hotels/popular
```
