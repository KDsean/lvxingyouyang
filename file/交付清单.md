## 🚀 快速启动

### 前置要求
- Java 17+
- Maven 3.6+
- MySQL 8.0+

### 启动步骤

#### 1. 初始化数据库
```bash
mysql -u root -p < init.sql
```

#### 2. 修改配置文件
编辑 `src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/lvxingyouyang
    username: root
    password: your_password
```

#### 3. 编译项目
```bash
mvn clean install
```

#### 4. 运行应用
```bash
mvn spring-boot:run
```

#### 5. 验证应用
访问 `http://localhost:8080/api/hotels/popular`

## 🔐 认证方式

所有需要认证的接口都需要在请求头中添加 JWT Token：

```
Authorization: Bearer <your_token>
```

### 获取 Token 示例
```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "password": "password123"
  }'
```

响应：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzUxMiJ9...",
    "user": {
      "id": 1,
      "username": "testuser",
      "email": "test@example.com",
      "phone": "13800138000",
      "avatar": null,
      "createdAt": "2026-03-20T10:00:00"
    }
  }
}
```
