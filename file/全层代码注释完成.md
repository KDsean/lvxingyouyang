## 🎯 注释标准

### Entity 注释标准
- 类级注释：说明实体对应的数据库表
- 字段注释：说明每个字段的含义、约束条件
- 回调方法注释：说明 @PrePersist、@PreUpdate 等方法的作用

### DTO 注释标准
- 类级注释：说明 DTO 的用途（请求/响应）
- 字段注释：说明每个字段的含义

### Controller 注释标准
- 类级注释：说明控制器的功能
- 方法注释：说明 API 端点的功能、参数、返回值

### Service 注释标准
- 类级注释：说明服务的功能
- 方法注释：说明方法的功能、参数、返回值、实现细节

### Config 注释标准
- 类级注释：说明配置类的功能
- 方法注释：说明配置方法的功能和作用

### Security 注释标准
- 类级注释：说明安全相关类的功能
- 方法注释：说明方法的功能和作用
